##  Project Overview
**DealDetective** is a Python-based application designed to help online shoppers find the **best deals** across multiple e-commerce sites.  
Using **web scraping**, **price comparison**, and **alert notifications**, it aims to save money for users who frequently shop online â€” especially students and parents looking for discounts on the same products.

---

##  Project Goals
- Compare prices across several e-commerce websites.  
- Track price changes and display historical price trends.  
- Allow users to set custom **price-drop alerts**.  
- Provide a simple, user-friendly interface for deal searching.
